  C#

C# is a simple, modern and general purpose object oriented language developed by Microsoft. This easy-to-use language becomes a powerhouse when combined with Visual Studio ecosystem

  LINQ

Language INtegrated Query (or more popularly 'LINQ') has revolutionized manipulating and interactinng with data. Even though, it is a part of C#, LINQ deserves a special mention because it has relieved programmers from worrying about underlying data structures. Throw a Objects, Datasets, XML or SQL, LINQ handles ALL.

  Entity Framework

Entity Framework (EF) is an open source, object-relational mapping (ORM) framework that supports the development of data-oriented software applications. The Entity Framework enables developers to work with data in the form of domain-specific objects and properties, without having to concern themselves with the underlying database tables and columns where this data is stored

  SQL

SQL is a special-purpose programming language designed for managing data held in a relational database management system (RDBMS). Nothing more to say, the name is enough.
